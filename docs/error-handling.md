# エラーハンドリング

## 形式

エラーは次の形式で返されます:

```json
{
  "error": {
    "message": "エラーメッセージ",
    "code": "エラーコード"
  }
}
```

## エラーコード一覧

| コード | 内容                  | エラーメッセージ                 |
| ------ | --------------------- | -------------------------------- |
| 400    | Bad Request           | リクエストの内容が不十分です。   |
| 401    | Unauthorized          | ログインしてください。           |
| 403    | Forbidden             | アクセス権限がありません。       |
| 404    | Not Found             | 指定したページが見つかりません。 |
| 500    | Internal Server Error | サーバーエラーです。             |
