## 1. ルーティング

| パス            | 説明                                      |
| --------------- | ----------------------------------------- |
| `/`             | メインページ（最新の移籍・怪我 5 件ずつ） |
| `/transfer`     | 移籍情報一覧（テーブル）                  |
| `/injury`       | 怪我情報一覧（テーブル）                  |
| `/transfer/:id` | 移籍情報                                  |
| `/injury/:id`   | 怪我情報                                  |
| `/login`        | ログイン・新規登録ページ                  |
| `/me`           | ユーザーページ                            |
